# 自动验证码识别小助手 - 极速安装版

你好！如果你需要自动填写网页上的字母数字图形验证码，只需要花 1 分钟照着做下面的两步，从此解套死记硬背！

## 第一步：给你的浏览器装个万能底座
不管你是用的谷歌 Chrome 还是微软 Edge 浏览器，请去他们各自的扩展应用商店搜索并安装：**篡改猴 (Tampermonkey)**
（你可以把它理解为一个能让你浏览器长出各种超级功能的免费工具）。

或者直接点击这里安装：[Tampermonkey 官网直接前往](https://www.tampermonkey.net/)

## 第二步：导入智能识别脚本
装好篡改猴之后，请把你的朋友发给你的 `auto_captcha.user.js` 这个脚本文件找出来。
你只需要：
1. **用鼠标左键按住这个 .js 文件，直接拖进你的浏览器窗口撒手**。
2. 浏览器会弹出一个“安装脚本”的绿色大按钮安全警告页面，直接点击【安装】即可。

**[操作成功]**
以后你在任何网站遇到需要填写验证码时，只要验证码图片旁边出现一个闪烁的【[一键识别填入]】按钮，用鼠标点它一下，验证码就会被隔空算出并自动跑到框里去了！就这么简单。
